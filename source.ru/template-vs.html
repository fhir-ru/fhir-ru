Пояснения к колонкам, которые могут появиться на этой странице:

УровеньНекоторые списки кодов могут быть иерархическими, где каждому коду присваивается некоторый уровень. По этой схеме некоторые коды оказываются под другими кодами, и подразумевается, что вложенные коды тоже применяются
ИсточникИсточник определения кода (если набор значений содержит коды, определённые где-то в другом месте)
КодКод (который используется в экземпляре ресурса)
Значение для отображенияЗначение для отображения (используется в элементе display в кодинге). Если значение для отображения не указано, реализаторам следует не просто показывать код, а использовать мэппинг для такого концепта в своём приложении
